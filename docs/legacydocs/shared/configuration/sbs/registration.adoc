== Registration

This document outlines the properties that participate or affect the registration process of the managed service in the Axile master.

include::../../glossary_link.adoc[leveloffset=+1]

=== Overview

When the Spring Boot application that has Axile Spring Boot Starter bootstraps, it tries to register itself in the Axile Master. Thus, the Axile SBS must provide certain information to the Axile SBS about itself. Some of this information may be inferred, but some needs to be explicitly set.

include::../common.adoc[leveloffset=+1]

== Usage Sample

YAML configuration example usage:

[source,yaml]
----
axile:
  sbs:
    register:
      instance-id: my-app-9bcf36fe-48579744
      master-url: http://master
      self-url: http://my-app:7075
----

The equivalent example for Java properties file:

[source,properties]
----
# The explicitly specified ID of that instance
axile.sbs.register.instance-id=my-app-9bcf36fe-48579744

# Master URL. Default port applied.
axile.sbs.register.master-url=http://master

# This service discoverable URL.
axile.sbs.register.self-url=http://my-app:7075
----
